接口特点：
1.同时提供银行卡在线支付、声讯电话支付、互联星空支付、手机短信注册、腾讯财付通、
腾讯Q币、神州行充值卡、盛大游戏点卡、支付宝、手机银行、北京宽带支付等支付途径。
2.银行卡在线支付支持国内60 余种银行卡（信用卡、储蓄卡、借记卡等）在线支付。
3.全国声讯电话支付支持全国所有省份固定电话、小灵通、中国移动、中国联通手机，
开通移动、联通、电信、网通多个声讯热线号码。
4.中国电信互联星空支付支持直接用163或ADSL上网帐号、各省互联星空网站注册用户支付。
5.支持中国移动、中国联通手机用户发送短信点播赠送服务。
6.开放腾讯公司Q币和财付通支付接口。
7.支持50元、100元、300元、500元等多种面值的神州行充值卡支付。
8.各支付途径可任意选择开启或关闭。
9.真正傻瓜式支付接口，提供安装程序、示例程序，只需简单设置即可使用。
10.设置商户密钥，支付信息加密传递，加强支付安全性，加密系统与其他支付平台兼容。
11.贺喜支付平台(http://www.168reg.cn)提供完善的后台管理系统，提供定单管理、定单
统计、财务管理、用户资料修改、商户密钥设置、推荐用户及技术支持等功能。
12.提交支付信息时可设置服务名称、商户订单号和两个自定义字段，支付成功后实时将支
付结果返回商户网站。
13.提供接口的测试模式，方便商户快捷地开发支付接口程序。
14.您可以登录后台管理系统查看定单以下信息：定单号、用户支付金额、商户所得金额、
自定义订单号、支付日期、结帐信息等。
15.提供简易版定单管理系统，具有定单查看、处理等功能。
16.提供宣传推广代码，推荐其他商户加盟可获得其收入的一定比例分成。
17.开放收费制商户，为商户降低支付成本。

使用说明：
一、文件列表：
setup.asp：支付接口安装程序。访问该文件可安装、设置支付接口。
index.asp：支付提交主程序。根据商户设置提交支付。
payment.asp：订单生成程序。内附详细说明和表单提交内容制作程序。
md5char.asp：函数文件。提供了MD5加密程序等函数。
payreceive.asp：订单支付成功返回验证页面。主要用于验证返回的订单信息是否正确。
paysettings.asp：支付接口设置文件。为支付接口设置商户号、商户密钥、返回地址等信息。
paysettings.mb：支付接口设置模板。
order文件夹：简易版定单查看、处理系统。
　├168regorder.asp：定单数据库，如需改名，请修改orderconn.asp文件设置。
　├index.asp：首页文件，自动判断并跳转至登录页面或定单查看页面。
　├login.asp：登录页面。
　├logout.asp：退出定单查看系统。
　├mysqlfire.asp：SQL防注入程序。
　├orderconn.asp：数据库连接程序，可修改定单数据库位置及文件名。
　├ordersec.asp：安全验证程序。
　├style.css：页面样式文件。
　├vieworder.asp：定单查看程序。
　└orderdispose.asp：定单处理程序。
“支付接口辅助工具”文件夹：提供MD5校验程序及源代码，可测试MD5加密字符串正确与否。

各文件具体说明和使用方法详见各文件头注释部分。

二、安装：
　　请将文件解压至同一目录中。
　　第一步：请先通过http://www.168reg.cn/注册成为商户。
　　第二步：获得商户编号，设置商户密钥。商户通过登录“贺喜支付平台”管理区，进入
“支付平台”－“参数设置”查看商户编号和设置商户密钥。
　　第三步：访问setup.asp安装支付接口，或用文本编辑器打开paysettings.asp文件，对
各项参数(变量)进行手工设置(详见文件中注释)。

三、测试：
　　安装时请选择“测试模式”，
安装完成后访问http://www.你的域名.com/168regpay/index.asp进行测试(此处假设支付接口
安装在网站168regpay文件夹中)。
　　测试成功后建议将支付接口集成到您的网站中并修改设置启用正式模式。
