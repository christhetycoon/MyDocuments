产品管理模块
功能 后台页面 图片和 flash 上传过程
分离出的前台页面 在根目录下的“ 产品前台”文件夹
  • ProductAdd.asp 在 iframe 中调用根目录下的 editor.asp
  • editor.asp里的 function pic() 调用 editor_insertpic.asp
  • editor_insertpic.asp 调用 editor_SelectUpFile.asp 选取 UploadFiles中的图片
  • editor_insertpic.asp 在 iframe 中调用 upload_dialog.asp 并传递参数
  • Upfile_Dialog.asp 把表单传给 Upfile_Dialog.asp
  • Upfile_Dialog.asp 调用上传组件完成上传。同时通过第 113，114 行命令将上传文件参数返回给 editor_insertpic.asp 里 Form1 的两个表单项目。
  • editor_insertpic.asp 利用 function OK() 把参数返回Editor.asp
  • Editor.asp 里的 function pic() 利用 parent.AddItem(ss[1]);使用ProductAdd.asp里的function AddItem(strFileName) 获取到图片参数。
  • ProductAdd.asp 把所有参数通过表单送往 ProductSave.asp 存储
  • Flash 的上传方式基本和图片相同
