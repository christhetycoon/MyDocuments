Flash Outline

A project I worked on during the weekend.

All data is read from a simple text file.
Search for: "outline.txt" in your browser's cache.

 

Peter Brouwers

 

 

Download flafile.

Read the explanation of the Text file below.

As a precaution two safety catches were built in:

- a check on the file being read completely [var AllDone ne ""]
- a check on whether the text file is there
