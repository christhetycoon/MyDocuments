The Cube Game

Assumed Knowledge:

  • Flash 4 Basics
  • Beginner's knowledge, really.

This isn't so much a game as it is an example for beginners. In order to learn what you can do with layers and 3D, must grasp the concept behind what makes something 3D. (ie. If it looks 3D, then it is ! )

 

 

 

 

 

Sorry, no real tutorial for this one. Although, the best way to learn is on your own, I find!

First, download the .FLA below ( flaped 9k). It should provide quite a bit of info for the learning beginner. Try to substitute symbols for different symbols. See what you can create! Cubes are a great device for simple(or complex) menus and interfaces, experiment!
