Generic Scroll Bar

by Bill Tagliaferro
[Download Source]

This tutorial is designed to show you how to build a generic scroll bar that can be used for most any application that requires scroll bar capability. It can be used both horizontally and vertically, and can be duplicated as many times as needed in a particular scene.

This tutorial is intended for intermediate Flash 4 users.

  • No previous tutorials are required.
  • Download the source files and have them unzipped for review during this tutorial.

Step-by-step

Overview:

Scroll bars have numerous applications. They often give a viewer control over what or how they view your movie elements. By building a generic scroll bar you will master the techniques for this project as well as have a scroll bar that can be added to your library of objects. The whole process can be summarized as follows:

  1. Building the images for the bar and scroller.

  2. Assign the actions to the scroller.

  3. Adding the scroll bar to a movie scene.

Background

A scroll bar in general will have a minimum and maximum value. These values are part of the overall design of your movie. In practice they might represent percentages, or scale factors, or numerical values etc. What we want here is a generic scroll bar so that we can use it for any application we want. To do this we will build a scroll bar that returns values from 0 to 1. In this case the general formula for scrolling will be:

movievalue = minimum + scrollvalue * (maximum - minimum)
The movievalue would then be used according to your design. For example it might set the alpha property, or the x/y scale property.



  1. Building the images for the bar and scroller.

    Keep in mind that whatever it is that you plan to drag (i.e. the scroller) will be a movie clip of a button. It will be actions associated with this button that do most of the work.

    1. The first step is to build a button to be used as the slider. For simplicity, just build a simple square button (CTRL + F8). Give it any name you like and don't worry about the button states.
    2. Decide how long your slide bar will be. It can be vertical or horizontal. It this tutorial I used a vertical bar 200 pixels long and the same width as the slide button.
    3. Create a new movie clip (CTRL + F8). Name this clip "SliderMovie".
    4. Drop an instance of your button into this clip and center it (CTRL + K).
    5. Double click on the button to open its Instance Properties window and click on the Action Tab.
    6. Assign the following actions to the button:

      On (Press)
      Start Drag ("", L=0, T=-100, R=0, B=100) End On
      On (Release, Release Outside)
      Stop Drag End On


      Key Points:
      1. In this tutorial I used a vertical bar so the Left (L) and Right(R) values are 0.
      2. The bar is going to be 200 pixels long and will be centered so the Top (T) value is -100 and the Bottom (B) is +100.
      3. You can always rotate the final Scroll Bar 90 degrees if you want a horizontal bar.


    7. Don't go on until you understand these actions or you will have difficulty with your final scroll bar.



  2. Assigning the actions to the scroller.
    1. Now create another new movie clip (CTRL + F8). Name this clip "ScrollBar".
    2. Using your drawing tools (Rectangle Tool) draw your slider bar 200 pixels heigh and the same width as your button.
    3. Center the bar (CTRL + K).
    4. Drop an instance of your "SliderMovie" into this clip and center it horizontally (CTRL + K) and place it vertically where you want it to start.
    5. Double click on the "SliderMovie" to open its Instance Properties window and click on the Definitions Tab.
    6. Make sure its behavior is "Movie Clip" and give it an Instance Name of "Slider".
    7. Insert an additional keyframe for a total of 2.
    8. Double click on the keyframe 1 to open its Frame Properties window and click on the Action Tab.
    9. Assign the following action to this frame:

      Set Variable: "scrollvalue" = (100 - slider:_y)/200

      Key Points:
      1. The bar I used was 200 pixels high and centered between -100 and +100.
      2. Scrollvalue will be a number between 0 and 1 depending on the Slider position.
      3. If your slide moves horizontally, then change _y to _x.


    10. Double click on the keyframe 2 to open its Frame Properties window and click on the Action Tab.
    11. Assign the following action to this frame:

      Go to and Play (1)



  3. Adding the scroll bar to a movie scene.
    1. Drop an instance of "ScrollBar" into your scene.
    2. Double click on the "ScrollBar" to open its Instance Properties window and click on the Definitions Tab.
    3. Make sure its behavior is "Movie Clip" and give it an Instance Name of "Scroll1".
      The value of the Scroll Bar will be located in "Scroll1:scrollvalue"
    4. YOU'RE DONE!. However, to see what you've done we will build a display to see your Scroll Bar in action.
      1. Using your tools add a textfield to your scene.
      2. Right click on this text field and give it a variable name of "display".
      3. Decide on the range of values you want to scroll between and pick a minimum and maximum.
      4. Double click on the keyframe 1 to open its Frame Properties window and click on the Action Tab.
      5. Assign the following action to this frame:

        Set Variable: "display" = scroll1:scrollvalue

Troubleshooting

The main problem you will have in this tutorial is remembering the paths to movie clips and variables. Try to use names that you will remember and have some association with the movie, instance or variable.
