Flash 4 - HitCounter

This Counter reads the value from a textfile or from given parameter (same swf-file).

1. Variant: Reading from file "counter.txt"
The file has to be in the same direktory, with the following contents:

--- File: counter.txt -----------------------------------------------------------
counter=01508
---------------------------------------------------------------------------------

2. Variant: Reading from given parameter
If you store the hits within a SQL-Database or use ASP.
Change the HTML code.
Object-Tag:

<PARAM NAME=movie VALUE="counter.swf?c=12345">
Embed-Tag:
<EMBED src="counter.swf?c=12345" ...
Remember to change both, object param AND embed !!!
