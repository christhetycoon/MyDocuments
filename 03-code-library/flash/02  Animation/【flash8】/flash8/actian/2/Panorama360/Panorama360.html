HOW TO MAKE A PANORAMIC VIEW SLIDER
(The join is intentionally messy so you can see how this works.)

 

 

There is no tutorial for this one, because the tutorial would be far too complicated. HOWEVER I always try to name variables names that make sense and place explanation comments liberally throughout the ".fla"
   This is an adaptation of the horizontal slider menu in the experts section of Virtual-FX. It is the exact same code as the sliding menu, just cut down to its basest minimum abilities. If you can't grasp the menu slider, download this and learn how it works first. That may make the menu much easier to understand!

This was made based on a request. (I normally don't do requests but since this was something that was sitting on the back shelf of things I always intended to get around to doing one day in Flash, here it is.)
