CSS
* <ul><li><a> 做菜单

* <ul><li> 的 list-style-type:none 去 bullet
* <ul><li> 的 display: inline + float：left 呈横排

* <a> display:block + width + height 定义按钮尺寸
* a:hover 定义 rollover style

show(node){
	this.active=node 不反应，退出
	this.active!=node, this.active!=null 马上隐藏 this.active 下面的 ul
	如果 node 下面有 ul，显示ul
	this.active = node;
	set_timeOut_events()
}

	flag = ture 马上隐藏

}
