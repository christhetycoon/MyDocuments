分离出的前台页面 在根目录下的“ 简介前台”文件夹
  • 从 admin/LeftTree.asp 传id 到 admin/About_modi.asp
  • About_modi.asp 在 iframe 中调用根目录下的 editor.asp 并把 Action=about&ID=<%=ID%>" 传给editor.asp
  • editor.asp 在 iframe 中调用根目录下的 ProductContent.asp 并把 Action=<%=trim(request("Action"))%>&ID=<%=trim(request("ID")) 传给 ProductContent.asp
  • ProductContent.asp 判断Action的类型并根据 id 把数据表 AboutUS 的内容读到在线编辑器中供修改
  • About_modi.asp 利用 function AddItem(strFileName) 读在线编辑器中的上传图片，利用 function CheckForm 读文本内容
  • About_modi.asp 把表单送给 about_save.asp。about_save.asp 根据id 判读并存储数据到表 AboutUs
