数值运算符

运算符 执行的运算 例子 说明
+ 加 a=8+3  
- 减 a=8-3  
* 乘 a=8*3  
/ 除 a=8/3  
% 求模 a=9%4 //a=1 求模就是计算余数，9除4的商是2，余数是 1。所以a的值是 1。
++ 递增 A++ A++ （先执行其它工作再递增）
++A （先递增再执行其它工作）
-- 递减 A--

逻辑运算符

逻辑运算符对两个布尔值进行比较并返回另一个布尔值

运算符 执行的运算 说明
&& 与 两个值都是 true 时，结果为 true
如果有一个 false，结果为 false
|| 或 两个值里有一个为 true，结果为 true
两个值都是 false，结果为 false
! 非 将 true 转为 false，将 false 转为 true

 

等于运算符

对操作数的值或者标识进行比较，返回布尔值。

运算符 执行的运算 说明
== 等于 123=="123" //true
=== 严格等于 123="123" // false  (不执行类型转换)
!= 不等于  
!== 严格不等于  

 

赋值运算符

运算符 执行的运算 例子 说明
= 赋值 a=5 a 的值被设为 5
+= 相加并赋值 a+=5 a 的值增加 5
-= 相减并赋值 a-=5 a 的值减少 5
*= 相乘并赋值 a*=5 a 的值乘 5
/= 相除并赋值 a/=5 a 的值除 5
%= 求模并赋值 a%=5 a 的值对 5 求模
<<= 向左移位并赋值 a<<=5 a 的值左移 5 位
>>= 向右移位并赋值 a>>=5 a 的值右移 5 位
>>>= 右移位填零并赋值 a>>>=5 a 的值右移 5 位填零
^= 取异或并赋值 a^=b a 与 b 取异或并赋值给 a
|= 取或并赋值 a|=b a 与 b 取或并赋值给 a
&= 取与并赋值 a&=b a 与 b 取与并赋值给 a

 

条件运算符

变量 = 条件 ? 结果1 ： 结果2

条件成立时，将 结果1 的值赋给变量。条件不成立时，将 结果2 的值赋给变量。
