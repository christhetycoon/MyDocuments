Install Bourbon

For rail2 or no-rails project

Install bourbon gem

gem install bourbon
gem install listen
gem install --version '~> 0.9.1' rb-fsevent

Release all library files into a folder called bourbon

bourbon install

Using watch command on project root to compile files

sass --watch stylesheets/sass:stylesheets -r ./stylesheets/bourbon/lib/bourbon.rb

CSS files structure

stylesheets
    bourbon
    sass
