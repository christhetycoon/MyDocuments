Draggable Make DOM elements draggable Draggable The jQuery UI Draggable plugin makes selected elements draggable by mouse.
Droppables To make an element react when a Draggable is dropped onto it Droppable The jQuery UI Droppable plugin makes selected elements droppable (meaning they accept being dropped on by draggables). You can specify which (individually) or which kind of draggables each will accept.
Sortable A Sortable is a quick way to initialize many Draggable elements in a container element. When you create a new Sortable, it takes care of the creation of the corresponding draggable Droppables. Sortable The jQuery UI Sortable plugin makes selected elements sortable by dragging with the mouse.
    Selectable The jQuery UI Selectable plugin allows for elements to be selected by dragging a box (sometimes called a lasso) with the mouse over the elements. Also, elements can be selected by click or drag while holding the Ctrl/Meta key, allowing for multiple (non-contiguous) selections.
    Resizable The jQuery UI Resizable plugin makes selected elements resizable (meaning they have draggable resize handles). You can specify one or more handles as well as min and max width and height.
