Select by tag name
document.getElementsByTagName() Accesses all elements with a specified tagname $$('tagName') Takes an arbitrary number of CSS selectors (strings) and returns a document-order array of extended DOM elements that match any of them. $("tagName") Selects all elements with the given tag name.

Select by event
    findElement() Returns the first DOM element that matches a given CSS selector — starting with the element on which the event occurred, then moving up its ancestor chain. If expression is not given, the element which fired the event is returned.    

Select by CSS selector
    select() Takes an arbitrary number of CSS selectors and returns an array of descendants of element that match any of them.    
    match() Checks if element matches the given CSS selector.    

Multi select
    $$ ("selector1, selector2, selectorN") return matching elements in array $("selector1, selector2, selectorN") Selects the combined results of all the specified selectors.
