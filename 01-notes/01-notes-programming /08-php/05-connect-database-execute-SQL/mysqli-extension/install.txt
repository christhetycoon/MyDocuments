Enabling the mysqli Extension on Linux/Unix
Enabling the mysqli extension on the Linux/Unix platform is accomplished by configuring PHP using the --with-mysqli flag. 
This flag should point to the location of the mysql_config program available to MySQL 4.1 and greater.
